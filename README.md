# HamburgueriaZ - Sistema de Pedidos Mobile

Aplicação desenvolvida como produto da aula prática da disciplina de **Desenvolvimento Mobile** do curso de **Engenharia de Software (UNOPAR/Anhanguera)**.

## 📱 Sobre o Projeto

O **HamburgueriaZ** é uma solução mobile intuitiva para automação de pedidos. O foco do projeto foi aplicar conceitos de reatividade de interface, persistência de estados temporários e integração entre aplicações via *Intents*.

## ✨ Funcionalidades

* **Cálculo em Tempo Real:** O valor total é atualizado instantaneamente ao selecionar adicionais ou alterar a quantidade.
* **Validação de Dados:** Sistema de proteção que impede pedidos sem nome ou com quantidade zero.
* **Integração com E-mail:** Geração automática de resumo estruturado enviado diretamente para o cliente de e-mail padrão.
* **Design Adaptativo:** Layout otimizado para telas com "notch" (câmara frontal) e suporte a modo ecrã inteiro (*Fullscreen*).

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java (JDK 21)
* **Interface:** XML com Material Design
* **IDE:** Android Studio
* **Ambiente:** Ubuntu Linux 24.04

## 🚀 Como Testar

Confira a versão compilada (**APK**) diretamente na seção de [Releases](https://github.com/mazinhorj/hamburgueriaz/releases/tag/v1.0.0) deste repositório.

1.  Baixe o arquivo `HamburgueriaZ_Mz.apk`.
2.  No seu Android, permita a instalação de **"Fontes Desconhecidas"**.
3.  Instale e faça o seu pedido!
